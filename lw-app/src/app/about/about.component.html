About Learn Word

A small application for building word collections and reviewing them without extra friction.

What it helps with

  • Create and manage word collections
  • Add words with translations and transcriptions
  • Review cards and track learning progress
  • Keep practice focused on the next word

Technology

  • Angular 17
  • Bootstrap 5
  • .NET Core Web API
  • Entity Framework Core
